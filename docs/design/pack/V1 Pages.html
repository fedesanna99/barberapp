01

Onboarding.

Il punto di ingresso · prima impressione del marchio · 4 stati del flusso di autenticazione.

01·a Welcome → prima apertura
Caricamento…

Schermata di benvenuto. Marchio grande, claim editoriale, due CTA dirette: Inizia (nuovo) e Accedi (esistente).

01·b Login → accesso esistente
Caricamento…

Email + password con valori demo precompilati. Link a password dimenticata e a registrazione.

01·c Registrati → nuovo account
Caricamento…

Selettore ruolo (cliente / barbiere), nome, email, password con indicatore di forza, captcha. Una sola colonna, scroll naturale.

01·d Recupera password → reset
Caricamento…

Una sola domanda: l'email. Bottone disabilitato finché non è valida. Copy diretta e gentile.

02

Cliente · golden path.

Le 4 schermate principali per chi vuole prenotare un taglio. Da Esplora si entra nel flusso di prenotazione in 2 tap.

02·a Esplora → home cliente
Caricamento…

Mappa + lista barbieri. Schermata di approdo del cliente dopo il login. Filtri orizzontali, pin discreti.

02·b Feed → social timeline
Caricamento…

Stories barbieri seguiti in alto, post 4:5 con caption sotto. Voce editoriale calmo.

02·c Profilo cliente → identità
Caricamento…

Avatar, stat strip, griglia tagli passati. Lo storico fa anche da portfolio personale.

02·d Menu → impostazioni
Caricamento…

Una sola schermata per tutte le scorciatoie: appuntamenti, notifiche, messaggi, supporto, tema.

03

Barbiere · lato professionista.

Il barbiere ha una tab in più ("Bottega") che è il suo dashboard. Atterra qui dopo il login.

03·a Bottega · oggi → dashboard
Caricamento…

Vista giornata: mono per i numeri (orari, prezzi, slot occupati), agenda visiva immediata. Punto di entrata del barbiere.

03·b Profilo barbiere → vetrina pubblica
Caricamento…

Come la vede il cliente che lo apre dalla mappa. Foto hero, stat strip, tab Lavori / Servizi / Recensioni, CTA fissa in fondo.

04

Sheet & overlay.

Le superfici che si sovrappongono allo schermo principale. Booking, appuntamenti, notifiche, messaggi.

04·a Booking sheet → il cuore del prodotto
Caricamento…

Servizio · data · ora · pagamento, tutto in un singolo sheet. Sticky summary in fondo: totale + CTA sempre visibili.

04·b I miei appuntamenti → overlay
Caricamento…

Prossimi e storia. Stato confermato / in attesa / cancellato in chip semantici. Bottoni rapidi per spostare o cancellare.

04·c Notifiche → overlay
Caricamento…

Prenotazione, like, follow, promemoria. Una linea per tipo, raggruppata per tempo. Icona discreta.

04·d Messaggi → overlay
Caricamento…

Chat dirette con i barbieri. Lista conversazioni + ultimo messaggio. Composer sotto per rispondere senza cambio schermo.

05

Admin.

Pannello di controllo interno. Validazione barbieri, segnalazioni, gestione contenuti.

05·a Admin → pannello
Caricamento…

Dashboard interno. Validazione profili barbiere in attesa, segnalazioni aperte, contatori live.
